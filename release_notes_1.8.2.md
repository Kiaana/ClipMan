# ClipMan v1.8.2 更新日志

> 本次更新专注于主题系统扩展和前端性能优化，带来更温馨的视觉体验和更流畅的交互。

## ✨ 新功能

- **主题系统扩展**：新增"淡粉色"主题，提供温馨柔和的视觉体验。
    - 采用低饱和度的"暖粉色"配色方案（Dusty Rose 风格）。
    - 新增独立的"外观"设置页面，提供直观的主题选择界面。
    - 支持在浅色、深色、淡粉色和跟随系统四种主题间切换。

## 🚀 改进与优化

### 性能提升

- **剪贴板项目组件优化**：重构了 `ClipboardItem` 组件的渲染逻辑。
    - 使用 Svelte 5 的 `$derived` 状态优化文本解码和图片处理，减少不必要的重新计算。
    - 改进了图片 Blob URL 的生命周期管理，防止内存泄漏。
    - 图片加载时显示"加载中"提示，提升用户体验。

### 用户体验 (UX)

- **复制操作反馈**：点击复制按钮后，图标会短暂变为绿色对勾，提供即时的视觉反馈。
- **平滑主题切换**：主题切换时背景色平滑过渡（300ms），避免突兀的颜色跳变。
- **列表动画**：置顶/取消置顶项目时，列表会以流畅的动画重新排序。
- **悬停效果增强**：
    - 剪贴板项目悬停时显示淡淡的背景色和左侧边框。
    - 操作按钮悬停时带有颜色和背景的过渡效果。
    - 图片悬停时会轻微放大（1.05倍），增强交互感。
- **二进制文件显示优化**：将"Binary File Data"改为更友好的中文提示，并添加图标。

## 🔧 内部 / 技术变更

- 重构了主题应用逻辑，确保 `light-pink` 类正确应用到 DOM 根元素。
- 优化了 `ClipboardItem` 组件的状态管理，使用 `$effect` 管理副作用。
- 为 `body` 元素添加了 `transition-colors` 类，实现全局平滑过渡。
